## 3. 구성 이유 (1-2)

### 마스터

- **Vive Tracker**: 서브mm 정확도, 120Hz, 기존 텔레옵 시스템에서 검증 완료. SteamVR 생태계 안정적
- **Manus Quantum Metagloves**: 20-DOF 이상 손가락 추적, 산업용 텔레옵에서 가장 많이 사용되는 장갑. SteamVR 플러그인 지원
- **5대 트래커 구성**: 양손(IK 타겟) + 허리(몸통 보정) + 양발(이동 제어). 최소 구성으로 전신 텔레옵 가능
- **Meta Quest Pro**: SteamVR 호환 + 패스스루로 조작자 시야 확보 가능. VR 피로도 감소

### 슬레이브

- **RB-Y1A**: 국산 휴머노이드, 기술 지원 접근성 우수. 2×7DoF 양팔 + 6DoF 몸통으로 dexterous manipulation에 적합. 모바일 베이스 포함
- **DG-5F**: 5-finger hand로 다양한 파지 형태 지원. RB-Y1A와 물리적/전기적 호환 확인 필요
- **RealSense D435i**: 소형, 저렴, ROS2 드라이버 성숙. 헤드 마운트로 조작자 시점 카메라 구현

### 통신

- **ROS2 Jazzy**: DDS 기반 실시간 통신, 마스터-슬레이브 분리 아키텍처에 적합
- **WiFi 6E (6GHz)**: 5ms 이하 지연, 영상 스트리밍과 명령 동시 전송

## 4. 백업 플랜 / 추가 구매 필요 항목 (1-3)

### 4-1. DG-5F 핸드 호환 이슈 시

**문제**: DG-5F가 RB-Y1A 엔드이펙터와 물리적으로 호환되지 않을 수 있음 (마운팅 플랜지, 전원 커넥터, 통신 프로토콜)

**백업**:
- **Option A**: 커스텀 마운팅 어댑터 제작 (기구 설계 + 3D 프린팅/CNC, ~200만 원)
- **Option B**: Inspire-Robots Dexterous Hand (대안 5-finger hand, ~2,000만 원/쌍)
- **Option C**: 초기에는 RB-Y1A 기본 그리퍼로 시작 → 핸드 별도 개발/구매 (일정 리스크)

### 4-2. HMD-카메라 지연이 VR 멀미 유발 시

**문제**: RealSense → ROS2 → 네트워크 → VR 렌더링 경로의 지연이 50ms 이상이면 조작자 불편

**백업**:
- **저지연 카메라 파이프라인 개발** (H.264 하드웨어 인코딩, WebRTC 기반, ~1개월 개발)
- **VR passthrough + AR 오버레이 방식**: HMD 패스스루 카메라를 기본 시야로 사용, 로봇 카메라는 PIP(Picture-in-Picture)로 표시 (SW 개발 필요)

### 4-3. Manus Glove 리드타임 / 가격 초과 시

**문제**: Manus Quantum 리드타임이 3개월+ 또는 가격이 예상 초과 가능

**백업**:
- **StretchSense MoCap Pro**: ~1,000만 원, 18-DOF, 정확도 다소 낮지만 빠른 납품
- **초기에는 Vive Tracker만으로 시작**: 손가락 없이 양손 위치/방향만 사용 → 그리퍼 개폐만 제어 (기능 축소)

### 4-4. RB-Y1A 납품 지연 시

**문제**: 로봇 본체 납품이 지연되면 10월 테스트베드 일정에 영향

**백업**:
- **MuJoCo 시뮬레이션 환경 활용**: 현재 구현된 MuJoCo + ROS2 bridge로 SW 개발 계속 진행 (HW 없이도 풀 스택 테스트 가능)
- **중간 검증**: 소형 로봇 암 (UR5e 등) 으로 마스터 시스템 및 IK 파이프라인 중간 검증

### 4-5. 추가 개발이 필요한 항목

| 항목 | 설명 | 예상 일정 |
|------|------|-----------|
| DG-5F 드라이버 개발 | ROS2 드라이버, 통신 프로토콜 구현 | 2-4주 |
| 핸드-로봇 마운팅 | 기구 설계 + 제작 | 2-3주 |
| 저지연 카메라 스트리밍 | H.264 인코딩 파이프라인 | 2-4주 |
| VR 렌더링 최적화 | 스테레오 렌더링, 왜곡 보정 | 2-3주 |
| 안전 시스템 | 물리 E-stop, 충돌 감지, 힘 제한 | 2-4주 |
| 테스트베드 환경 셋업 | 작업 공간 구성, 네트워크, 전원 | 1-2주 |

## 6. 논의 필요 사항

1. **RB-Y1A 납품 일정 확인** — 10월 테스트베드 기준 최소 7월 초 납품 필요
2. **DG-5F 호환성 확인** — Rainbow Robotics 측 기구/전기 인터페이스 스펙 확인
3. **Manus Glove 견적/리드타임 확인** — 정확한 가격, 납품 기간 문의
4. **테스트베드 공간 요구사항** — Base Station 설치 가능 공간 (최소 4m×4m), 전원, 네트워크
5. **안전 기준** — 조작자/주변인 안전을 위한 물리 E-stop 설치 위치, 힘 제한 기준
